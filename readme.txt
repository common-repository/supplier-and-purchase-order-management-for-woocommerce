Manage suppliers and create purchase orders from your WooCommerce products

== Description ==
Manage your suppliers, create purchase orders, receive deliveries and show current inventory, all from within one Plugin. This plugin uses your WooCommerce Products which can be assigned to a supplier. It is possible that several suppliers can be assigned the same products. This plugin also uses your existing WooCommerce design to generate confirmation emails and POs in the style of your existing store emails.

== Installation ==
Download the plugin, install and activate.

== Frequently Asked Questions ==

= How to use the plugin? =

1. step - Add your supplier(s)
2. step - Assign woocommerce products to your supplier
3. step - create and manage your purchase orders for the existing suppliers

= I receive the same product from different suppliers. Can I assign the same product to multiple suppliers? =

Yes, unlike other plugins, you can assign the same product to several suppliers.

= I would like to order without prices only by quantities. Is that possible? =

Yes, it is possible only to add the quantities when creating a purchase order. Prices and fields for tax are optional.
